CSS Font Families

In CSS, there are two types of font family names:

  • generic family - a group of font families with a similar look(like 'Serif' or 'Monospace')
  • font family - a specific font family(like "Times New Roman" or "Arial")

PropertyDescription
fontSets all the properties in one declaration
font-familySpecifies the font family for text
font-sizeSpecifies the font style for text
font-variantSpecifies whether or not a text should be displayed in a small-caps font
font-weightSpecifies the weight of a lot
