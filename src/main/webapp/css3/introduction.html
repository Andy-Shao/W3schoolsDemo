CSS3 Modules

CSS3 has been split into "modules". It contains the "old CSS specification" (which has been split smaller pieces). In addition, new modules are added.

Some of the most important CSS3 modules are:

  • Selector
  • Box Model
  • Backgrounds and Borders
  • Image Values and Replaced Content
  • Text Effects
  • 2D/3D Transformations
  • Animations
  • Multiple Column Layout
  • User Interface
