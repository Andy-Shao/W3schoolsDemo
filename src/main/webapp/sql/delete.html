The SQL DELETE Statement

The DELETE statment is used to delete rows in a table.

SQL DELETE Syntax

DELETE FROM table_name WHERE some_column=some_value;

Notice the WHERE clause in the SQL DELETE statement!

The WHERE clause specifies which record or records that should be deleted. If you omit the WHERE clause, all records will be deleted!

Delete All Data

It is possible to delete all rows in a table without deleting the table. This means that the table structure, attribute, and indexes will be intact:

DELETE FROM table_name
or
DELETE * FROM table_name

Note: Be very careful when deleting records. You cannot undo this statement!
